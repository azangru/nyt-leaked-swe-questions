Parenthesis Matching
====================

Given a string composed of the characters `(` and `)`, implement an algorithm to
determine whether everything is balanced. Example data:

| Pattern    | Valid |
| ---------- | ----- |
| `"()(())"` | yes   |
| `"()(())"` | yes   |
| `"())("`   | no    |
| `"(()"`    | no    |
