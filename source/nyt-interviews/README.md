# A collection of take home tests used across teams at NYT

This repo is a collection of the various take home tests used across teams at the New York Times. If you already have a take home test in the form of a github repo, please add a link to this readme. Alternatively if you wish your test to be a part of this repo, add a markdown file with the problem and link it to this README.

- [Delivery Engineering take home test](https://github.com/nytm/dv-interview)
- [Knowledge Management Take Home](https://github.com/nytm/knowmg-interview-exercise)
- [Search team programming test](https://github.com/nytm/search/tree/master/hiring/test)
- [Data Science homework](https://github.com/nytm/dse-experiments/tree/master/shared/homework/homework_singlecopy)
- [NYT News Products Code Test](https://github.com/nytm/news-products-interview)
- [NYT Data Engineering Homework](nyt-data-engineering-homework.md)
- [Games iOS Code Test](https://github.com/nytm/ios-code-test-swift)
